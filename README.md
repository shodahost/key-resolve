# Skróty DaVinci Resolve — ściąga

Statyczna strona ze skrótami klawiszowymi DaVinci Resolve. Jeden plik `index.html`, bez zależności i builda.

## Funkcje
- 🔍 Wyszukiwarka skrótów (klawisz `/` ustawia kursor, `Esc` czyści)
- 🏷️ Filtrowanie po kategoriach
- 🌗 Tryb jasny / ciemny (zapamiętywany)
- 📱 Responsywny układ
- ⚡ Sekcja J/K/L i Top 5 na start

> Strona działa też po prostu otwarta lokalnie w przeglądarce (dwuklik na `index.html`).
